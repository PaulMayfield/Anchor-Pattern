# GitHub Pages Setup Instructions

Your Anchor Pattern game is ready! Here's how to enable GitHub Pages:

## Quick Setup

1. Go to: https://github.com/PaulMayfield/Anchor-Pattern/settings

2. Scroll down to the "Pages" section (left sidebar)

3. Under "Source", select:
   - Branch: **main**
   - Folder: **/ (root)**

4. Click "Save"

5. GitHub will show you a URL like: `https://paulMayfield.github.io/Anchor-Pattern/`

## Your Game URL

Once GitHub Pages is enabled, your game will be live at:
`https://paulMayfield.github.io/Anchor-Pattern/`

You can share this link with testers immediately!
